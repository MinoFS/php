Variables et types de données

data_object Variables en PHP

• Toujours précédées du symbole $

• Sensible à la casse : $nom ≠ $Nom

• Déclaration et initialisation en une seule étape

• PHP est un langage à typage dynamique

category Types de base

string
Chaîne de caractères
int
Nombre entier
float
Nombre décimal
bool
Vrai ou faux
tips_and_updates Points importants
check_circle PHP détermine automatiquement le type de la variable
check_circle Une variable peut changer de type pendant l'exécution
check_circle Utilisez var_dump() pour vérifier le type d'une variable

Exemples de variables et types

<?php

// Déclaration de variables de différents types
$nom = "Dupont"; // string
$age = 25; // int
$prix = 19.99; // float
$estEtudiant = true; // bool

// Affichage des variables
echo "Nom : " . $nom . "<br>";
echo "Âge : " . $age . " ans<br>";
echo "Prix : " . $prix . " €<br>";
echo "Étudiant : " . $estEtudiant . "<br>";

// Changement de type (typage dynamique)
$age = "vingt-cinq"; // maintenant un string

// Vérification du type
var_dump($nom);
var_dump($age);

?>
