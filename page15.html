Erreurs courantes chez les débutants

error_outline Oublier le symbole $

• Toutes les variables PHP doivent commencer par $

• Erreur fréquente quand on vient d'autres langages

cancel Incorrect
<?php
$nom = "Dupont";
echo nom; // Erreur!
?>
check_circle Correct
<?php
$nom = "Dupont";
echo $nom; // OK
?>

error_outline Oublier le point-virgule

• Chaque instruction PHP doit se terminer par un ;

• Une des erreurs les plus fréquentes

cancel Incorrect
<?php
$age = 25 // Manque ;
echo "Bonjour" // Manque ;
?>
check_circle Correct
<?php
$age = 25;
echo "Bonjour";
?>

error_outline Confondre = et ==

• = : opérateur d'affectation (assigne une valeur)

• == : opérateur de comparaison (teste l'égalité)

• === : comparaison stricte (valeur ET type)

cancel Incorrect
<?php
$age = 18;
if ($age = 18) { // Toujours vrai!
  echo "Majeur";
}
?>
check_circle Correct
<?php
$age = 18;
if ($age == 18) { // Teste l'égalité
  echo "Majeur";
}
?>
