Boucles

loop Types de boucles

• Permettent d'exécuter du code plusieurs fois

• Évitent la répétition de code

repeat Boucle for

• Répéter un nombre défini de fois

• Utilise un compteur

for (initialisation; condition; incrémentation) {
  // Code à exécuter
}

sync Boucle while

• Répéter tant qu'une condition est vraie

• Nombre d'itérations inconnu

while (condition) {
  // Code à exécuter
}

lightbulb Exemple pédagogique

Créer une table de multiplication ou afficher une liste d'éléments

Exemples de boucles

<?php

// Exemple de boucle for : table de multiplication
$table = 5;
echo "Table de multiplication de $table :<br>";

for ($i = 1; $i <= 10; $i++) {
  $resultat = $table * $i;
  echo "$table × $i = $resultat<br>";
}

// Exemple de boucle while : décompte
$compteur = 5;
echo "<br>Décompte :<br>";

while ($compteur > 0) {
  echo "$compteur...<br>";
  $compteur--; // Décrémentation
}
echo "Fin !<br>";

// Exemple de boucle foreach : liste de fruits
$fruits = array("Pomme", "Banane", "Orange", "Fraise");
echo "<br>Liste de fruits :<br>";

foreach ($fruits as $fruit) {
  echo "- $fruit<br>";
}

// Exemple pratique : afficher une liste HTML
$etudiants = array("Alice", "Bob", "Charlie", "David");
echo "<br><ul>";
foreach ($etudiants as $etudiant) {
  echo "<li>$etudiant</li>";
}
echo "</ul>";

?>
