Bonnes pratiques de base

format_align_left Indentation

• Utilisez une indentation cohérente (4 espaces ou tabulation)

• Indentez à l'intérieur des structures de contrôle

• Améliore la lisibilité du code

check_circle Bon
if ($condition) {
  // Code indenté
  $result = $a + $b;
}
cancel Mauvais
if ($condition) {
// Code non indenté
$result = $a + $b;
}

code Séparer logique et affichage

• Traitez les données en haut de page

• Affichez les résultats en bas de page

• Évitez de mélanger traitement et HTML

Exemple de bonne structure

<?php
// Traitement des données (logique)
$nom = "";
$message = "";
$erreur = "";

if (isset($_POST['submit'])) {
  $nom = isset($_POST['nom']) ? $_POST['nom'] : "";
  $message = isset($_POST['message']) ? $_POST['message'] : "";

  if (empty($nom) || empty($message)) {
    $erreur = "Veuillez remplir tous les champs";
  } else {
    // Traitement des données...
    $succes = "Message envoyé avec succès";
  }
}
?>

// Affichage (HTML)
<!DOCTYPE html>
<html lang="fr">
<head>
  <meta charset="UTF-8">
  <title>Formulaire</title>
</head>
<body>
  <h1>Formulaire</h1>

  <?php if (!empty($erreur)) { ?>
    <div class="erreur"><?= $erreur ?></div>
  <?php } ?>

  <?php if (!empty($succes)) { ?>
    <div class="succes"><?= $succes ?></div>
  <?php } ?>

  <form method="POST" action="">
    <input type="text" name="nom" value="<?= $nom ?>">
    <textarea name="message"><?= $message ?></textarea>
    <button type="submit" name="submit">Envoyer</button>
  </form>
</body>
</html>
