PHP et HTML ensemble

integration_instructions Intégration de PHP dans HTML

• PHP peut être intégré directement dans du code HTML

• Le code PHP est exécuté sur le serveur avant d'envoyer le HTML

• L'extension du fichier doit être .php (pas .html)

code Syntaxe d'intégration

• <?php ... ?> : Bloc de code PHP

• <?= $variable ?> : Raccourci pour echo

• PHP peut être placé n'importe où dans le HTML

visibility Affichage de variables PHP

check_circle Utilisez echo pour afficher du contenu
check_circle Concaténez avec l'opérateur . (point)
check_circle Intégrez dans les attributs HTML ou le contenu
tips_and_updates Points importants
check_circle Le code PHP n'est jamais visible par le navigateur
check_circle Le résultat HTML est généré dynamiquement

Exemple d'intégration PHP/HTML

<?php
// Définition des variables
$titre = "Ma page PHP";
$utilisateur = "Jean Dupont";
$couleur = "#4285F4";
$produits = array("Livre", "Stylo", "Cahier");
?>

<!DOCTYPE html>
<html lang="fr">
<head>
  <meta charset="UTF-8">
  <title><?= $titre ?></title>
  <style>
    body { font-family: Arial, sans-serif; }
    .header { background-color: <?= $couleur ?>; color: white; padding: 20px; }
  </style>
</head>
<body>
  <div class="header">
    <h1>Bonjour <?= $utilisateur ?>!</h1>
  </div>
  <div class="content">
    <h2>Liste de produits</h2>
    <ul>
<?php
    // Boucle pour afficher les produits
    foreach ($produits as $produit) {
      echo "<li>$produit</li>\n";
    }
?>
    </ul>
  </div>
</body>
</html>
