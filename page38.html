Erreurs courantes débutants

account_tree

Confusion base/table

• Base de données : conteneur principal

• Table : structure pour stocker des données similaires

Erreur fréquente

SELECT * FROM blog;
// Erreur: 'blog' est une base, pas une table
lightbulb

Utilisez base.table pour spécifier la table : blog.messages

code

Erreurs de syntaxe SQL

• Oubli de guillemets pour les chaînes

• Oubli de virgules entre les colonnes

• Mauvais ordre des instructions

Erreur fréquente

INSERT INTO messages (auteur, message) VALUES (Alice, Bonjour);
// Erreur: guillemets manquants
lightbulb

Entourez les chaînes de guillemets simples : 'Alice'

link_off

Problème de connexion

• Mauvais identifiants de connexion

• Serveur MySQL non démarré

• Base de données inexistante

Erreur fréquente

$pdo = new PDO("mysql:host=localhost;dbname=blog", 'root', 'password');
// Erreur: mot de passe incorrect
lightbulb

Vérifiez les identifiants et assurez-vous que MySQL est démarré

error_outline

Autres erreurs courantes

• Noms de tables/champs incorrects

• Types de données incompatibles

• Caractères spéciaux non échappés

Erreur fréquente

SELECT * FROM Messages;
// Erreur: la table s'appelle 'messages' (minuscule)
lightbulb

Respectez la casse et utilisez des requêtes préparées
