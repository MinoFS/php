• empty() : vérifie si une variable est vide

• Retourne true si : "", 0, "0", null, false, array()

• isset() : vérifie si une variable existe

lightbulb Utilisation avec un formulaire

Vérifier si un champ a été rempli avant de traiter les données

tips_and_updates Bonne pratique

Toujours vérifier si les variables d'un formulaire existent avec isset() avant de les utiliser

Exemple de conditions

<?php

// Vérification de l'âge
$age = 20;

if ($age >= 18) {
  echo "Vous êtes majeur.<br>";
} else {
  echo "Vous êtes mineur.<br>";
}

// Vérification avec elseif
$note = 15;

if ($note >= 16) {
  echo "Très bien<br>";
} elseif ($note >= 14) {
  echo "Bien<br>";
} elseif ($note >= 12) {
  echo "Assez bien<br>";
} else {
  echo "À améliorer<br>";
}

// Vérification si une variable est vide
$nom = "";

if (empty($nom)) {
  echo "Le nom est vide.<br>";
} else {
  echo "Bonjour " . $nom . "!<br>";
}

// Exemple avec un formulaire
if (isset($_POST['submit'])) {
  $message = $_POST['message'];
  
  if (empty($message)) {
    echo "Veuillez saisir un message.<br>";
  } else {
    echo "Message reçu : " . $message . "<br>";
  }
}

?>
