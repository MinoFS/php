Premier script PHP

code Balises PHP

• Code PHP toujours entre <?php ?>

• Permet de différencier le code PHP du HTML

• Peut être inséré n'importe où dans un fichier HTML

output Instruction echo

• echo : affiche du texte ou des variables

• Équivalent de "print" ou "console.log"

• Génère du code HTML envoyé au navigateur

tips_and_updates Points importants
check_circle Toutes les instructions PHP se terminent par un point-virgule (;)
check_circle Les fichiers PHP doivent avoir l'extension .php
check_circle Le code PHP est exécuté sur le serveur, jamais dans le navigateur

Exemple "Hello World"

// Fichier : hello.php
<?php

// Affichage simple de texte
echo "Hello World!";

// Affichage avec balises HTML
echo "<h1>Bonjour</h1>";

// Affichage avec retour à la ligne
echo "Bienvenue en PHP<br>";
echo "C'est votre premier script!";

?>
visibility Résultat dans le navigateur
Hello World!

Bonjour


Bienvenue en PHP
C'est votre premier script!
