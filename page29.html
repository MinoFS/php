SQL — bases indispensables : INSERT

add_circle Commande INSERT

• Permet d'ajouter des données dans une table

• Structure de base : INSERT INTO table VALUES (...)

• Essentielle pour stocker de nouvelles informations

code Exemples de INSERT

INSERT INTO messages VALUES (0, 'Alice', 'Bonjour !', NOW());

Ajoute un nouveau message avec toutes les valeurs

INSERT INTO messages (auteur, message) VALUES ('Bob', 'Merci !');

Ajoute un message en spécifiant certaines colonnes

INSERT INTO utilisateurs (nom, email, age) VALUES ('Charlie', 'charlie@email.com', 25);

Ajoute un nouvel utilisateur dans la table

lightbulb Points clés
check_circle L'ordre des valeurs doit correspondre à l'ordre des colonnes
check_circle Les chaînes de caractères doivent être entre 'guillemets simples'
check_circle NOW() insère la date et heure actuelles

Exemple complet avec INSERT

-- Insérer un message complet
INSERT INTO messages VALUES (
NULL, -- ID (auto-incrémenté)
'Alice', -- Auteur
'Bonjour à tous !', -- Message
NOW() -- Date actuelle
);

-- Insérer en spécifiant les colonnes
INSERT INTO messages (auteur, message) VALUES (
'Bob',
'Merci pour l\\'information'
);

-- Insérer plusieurs messages en une fois
INSERT INTO messages (auteur, message) VALUES
('Charlie', 'Super cours !'),
('David', 'J\\'apprends beaucoup'),
('Eve', 'Très intéressant');

-- Insérer un utilisateur
INSERT INTO utilisateurs (nom, email, age) VALUES (
'Alice Dupont',
'alice@email.com',
22
);
