SQL — bases indispensables : UPDATE

edit Commande UPDATE

• Permet de modifier des données existantes

• Structure de base : UPDATE table SET colonne=valeur WHERE condition

• Attention : sans WHERE, toutes les lignes sont modifiées !

code Exemples de UPDATE

UPDATE messages SET message = 'Nouveau message' WHERE id = 1;

Modifie le message avec l'ID 1

UPDATE utilisateurs SET age = 26 WHERE nom = 'Alice';

Modifie l'âge de l'utilisatrice Alice

UPDATE messages SET auteur = 'Alice D.' WHERE auteur = 'Alice';

Modifie tous les messages de l'auteur 'Alice'

warning

Attention : Oublier la clause WHERE peut modifier toutes les lignes de la table !

lightbulb Points clés
check_circle Toujours utiliser une clause WHERE précise
check_circle Peut modifier plusieurs colonnes en une seule commande

Exemple complet avec UPDATE

-- Mettre à jour un message spécifique
UPDATE messages
SET message = 'Bonjour à tous ! (modifié)'
WHERE id = 1;

-- Mettre à jour plusieurs colonnes
UPDATE utilisateurs
SET
  nom = 'Alice Dupont',
  email = 'alice.dupont@email.com'
WHERE id = 1;

-- Mettre à jour plusieurs lignes
UPDATE messages
SET auteur = 'Anonyme'
WHERE auteur = 'Bob';

-- Mettre à jour avec une condition complexe
UPDATE messages
SET message = '[Ancien] ' || message
WHERE date_creation < '2023-01-01'
AND auteur = 'Alice';

-- Augmenter l'âge de tous les utilisateurs
UPDATE utilisateurs
SET age = age + 1;
