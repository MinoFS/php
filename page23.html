forum Exemple concret : Livre d'or

Sans base de données :

• Messages stockés dans un fichier texte

• Difficile de rechercher un message spécifique

• Risque de corruption du fichier

• Pas de tri possible

Avec MySQL :

• Messages structurés (auteur, contenu, date)

• Recherche facile par auteur ou contenu

• Tri par date automatique

• Gestion des doublons
