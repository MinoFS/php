Qu'est-ce que phpMyAdmin ?

• Interface web pour gérer MySQL

• Permet de créer, modifier et supprimer des bases de données

• Inclu dans XAMPP et WAMP

• Alternative à la ligne de commande

1

Créer une base de données

Onglet "Bases de données" → Nom de la base → "Créer"

2

Créer une table

Sélectionner la base → Nom de la table → Nombre de colonnes → "Exécuter"

3

Définir les colonnes

Nom, Type, Taille, etc. → "Sauvegarder"

4

Insérer des données

Onglet "Insérer" → Remplir les champs → "Exécuter"
