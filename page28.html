SQL — bases indispensables : SELECT

visibility Commande SELECT

• Permet de lire des données dans une table

• La commande la plus utilisée en SQL

• Structure de base : SELECT colonnes FROM table

code Exemples de SELECT

SELECT * FROM messages;

Affiche toutes les colonnes de la table messages

SELECT auteur, message FROM messages;

Affiche uniquement les colonnes auteur et message

SELECT * FROM messages WHERE auteur = 'Alice';

Affiche les messages où l'auteur est Alice

SELECT * FROM messages ORDER BY date_creation DESC;

Affiche les messages triés par date décroissante

lightbulb Points clés
check_circle * signifie "toutes les colonnes"
check_circle WHERE permet de filtrer les résultats
check_circle ORDER BY trie les résultats (ASC ou DESC)

Exemple complet avec SELECT

-- Sélectionner tous les messages
SELECT * FROM messages;

-- Sélectionner uniquement certains champs
SELECT id, auteur, message FROM messages;

-- Filtrer avec WHERE
SELECT * FROM messages WHERE auteur = 'Alice';

-- Trier les résultats
SELECT * FROM messages ORDER BY date_creation DESC;

-- Limiter le nombre de résultats
SELECT * FROM messages LIMIT 5;

-- Combiner plusieurs conditions
SELECT * FROM messages
WHERE auteur = 'Alice' AND date_creation > '2023-01-01'
ORDER BY date_creation DESC LIMIT 10;
