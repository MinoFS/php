Installation et environnement

dns Principe serveur web

• Serveur web : programme qui répond aux requêtes HTTP

• Apache : serveur web le plus utilisé avec PHP

• PHP s'exécute uniquement sur le serveur

• Le navigateur reçoit uniquement le résultat HTML

get_app Installation avec XAMPP/WAMP

• XAMPP : Cross-platform Apache MySQL PHP Perl

• WAMP : Windows Apache MySQL PHP

• Packages tout-en-un pour développer en local

• Installation simple en quelques clics

lightbulb Conseil
Utilisez XAMPP si vous travaillez sur Windows, macOS ou Linux. WAMP est spécifique à Windows.
